### card.queue_stack
#### 卡片: 队列 & 栈

*  BFS 广度优先搜索 
主要用于 __最短路径__ 与 __遍历所有节点__ 
1. 套路就是找到题目的起始状态跟终点状态。  
2. 然后通过一个队列记录所有的状态, 每次遍历队列所有节点。同时路径加1。   
3. 遍历节点时取出第一个节点后删除, 对这个节点进行邻居节点遍历并加入到队列中。     
    3.1 在这里首先检测这个节点值是否是目标, 是就结束。               
    3.2 求最短路径的邻居节点就是分析这个搜索是如何分支开来的。  
    3.3 一般情况都会利用一个集合记录已经遍历过的节点(相同的结果)。
